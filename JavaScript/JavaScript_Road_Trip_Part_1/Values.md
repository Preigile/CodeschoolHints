## 1.2 Базовые операции

_Задание._
В консоль ввести любое выражение, используя +, -, *, или /, так, чтобы ответ был равен 36.

_Решение._
Вспоминая таблицу умножения:
```
6 * 6
```

_Пояснение._
Базовые операции в JavaScript точно такие, как в обычной математике.
