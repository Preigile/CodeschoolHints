# Резюме

*Операторы:* + (сложение), - (вычитание), * (умножение), / (деление), % (остаток от деления);   
*Сравнение:* < (меньше), > (больше), <= (меньше или равно), >= (больше или равно), == (равно), != (не равно), === (строгая равность);
*Специальные символы:* \\ (\), \n (новая строка), \t (таб);
*=* - присвоение;
